API Documentation

Get Subscribers

Endpoint: /subscribers

Method: GET

Response: The API will respond with an array of JSON objects containing the following fields.

  • _id: Unique id of Subscriber.
  • name: Name of Subscriber.
  • subscribedChannel: Name of channel.
  • subscribedDate: Date of Subscription.
    [
        {
        "_id": "6651e04bb2ae8d06e3df10e8",   
        "name": "Jeread Krus",
        "subscribedChannel": "CNET",
        "subscribedDate": "2024-05-25T12:57:47.677Z"
        },
        {
        "_id": "6651e04bb2ae8d06e3df10e9",
        "name": "John Doe",
        "subscribedChannel": "freeCodeCamp.org",
        "subscribedDate": "2024-05-25T12:57:47.677Z"
        },
    ]

Get Subscribers Names

Endpoint: /subscribers/names

Method: GET

Response: The API will respond with an array of JSON objects containing the following fields.

  • name: Name of Subscriber.
  • subscribedChannel: Name of channel.
    [
        {
        "name": "Jeread Krus",   
        "subscribedChannel": "CNET"
        },
        {
        "name": "John Doe",
        "subscribedChannel": "freeCodeCamp.org"
        },
    ]

Get Subscriber by ID

Endpoint: /subscribers/:id

Method: GET

Parameter: - ID of the subscriber

Response: The API will respond with an array of JSON objects containing the following fields.

  • _id: Unique id of Subscriber.
  • name: Name of Subscriber.
  • subscribedChannel: Name of channel.
  • subscribedDate: Date of Subscription.
